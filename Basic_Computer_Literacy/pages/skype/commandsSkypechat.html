Какие команды чата доступны в Skype

Чтобы увидеть список доступных команд, просто введите /help в чат. Если вы находитесь в чате только с одним человеком, будут показаны только соответствующие параметры чата. Если вы общаетесь в чате с двумя или более людьми, /help покажет более полный список команд чата. Ниже приведен полный список доступных команд чата в Скайпе и их описания.

/me [text] - Ваше имя появится после любого текста, который вы напишете. Например, /me работаю дома приведет к тому, что рядом с вашим именем в чате появится фраза «работаю дома». Вы можете использовать это, чтобы отправить сообщение о вашей деятельности или статусе.

/poll [title], [option1], [option2], [option3] - Создайте вопрос для опроса, который будет отправлен в ваш чат. Введите вопрос через запятую, затем через запятую укажите варианты ответа.

/leave - Покиньте текущий групповой чат.

/showmembers - Перечисляет всех участников чата с указанием их ролей.

/setrole [Skype Name] [Admin/User] - Позволяет установить роль каждому участнику чата. Роль администратора (Admin) — это организатор чата Skype, он может добавлять или удалять участников, настраивать параметры чата и назначать других людей администраторами. Пользователь (User) – участник, который может публиковать сообщения в чате и участники звонков. Все участники группового чата изначально являются администраторами, за исключением модерируемого чата. Примечание. В модерируемой группе только создатель чата может устанавливать роли.

/golive - Начинает групповой звонок с другими участниками чата. Примечание. В модерируемой группе только создатель чата имеет возможность /goliv.

/clearmru - Очистить список недавно использованных элементов самовыражения.

/version - Показывает текущую версию Skype, в которой вы используете.

/addreaction [Emoticon name] - Добавляет реакцию в средство выбора реакции на сообщение. Например, /addreaction like добавит к вашим реакциям смайлик «Нравится». Вы также можете настроить средство выбора реакции на сообщение.

/addreactions [Emoticon name] - Добавляет реакции в средство выбора реакций на сообщение.

/removereaction [Emoticon name] - Удаляет реакцию из средства выбора реакции на сообщение.

/removereactions [Emoticon name] - Удаляет реакции из средства выбора реакций на сообщение.

/resetreactions - Сбрасывает реакции, установленные по умолчанию.

/language [ll-CC|ll|reset] - Изменить язык.

/openinext [on|off] - Включить открытие ссылок во внешнем браузере.

/formatting [on|off|reset|resetall] - Смайлик и специальное форматирование исходящих сообщений (для каждого чата).

/addcallreaction [emoticon name string] - Добавить реакцию на вызов.

/addcallreactions [emoticon name string] - Добавить реакции на вызов.

/printcallreactions (optional) position - Отображает доступные реакции на вызов.

/removecallreactions[emoticon1], [emoticon2], … - Удаляет реакции на вызов.

/removecallreactionsat (reaction positions, use /printcallreactions for more info) - Удаляет реакции на вызовы в указанной позиции.

/resetcallreactions - Сбрасывает реакции на вызовы, установленные по умолчанию.

/showbrowserinfo - Показывает информацию о браузере.

Команды администратора модерируемой группы:

/kick [Skype Name] - Удаление участника чата.

/kickban [Skype Name] - Удаление участников чата и запретить им повторно присоединяться к модерируемому чату. Запрет постоянный.

/getbanlist - Перечисляет всех людей, забаненных в модерируемом чате.

/setbanlist [(+)|-] [Skype Name] - Укажите, кому из участников будет запрещен доступ в модерируемый чат.
